# roms 目录说明

本目录按来源仓库名分类存放 ROM。当前仅内置 `nes-open-db` 的全量 7 个 Homebrew ROM，不内置和分发商业 ROM。

当前结构：

- `roms/nes-open-db/`：ROM 文件
- `roms/licenses/`：许可证文本

白名单清单见 `roms/WHITELIST.md`，对应许可证文本位于 `roms/licenses/`。

如果你要游玩其他 ROM，请在页面中使用“本地导入 ROM”功能，选择你已合法获得授权的 `.nes` 文件。

任何新增到本目录的 ROM 都必须同时提交，并按 `roms/<来源仓库名>/` 归档：

1. 来源链接
2. 明确许可证或授权文本
3. 授权核验日期
