## 后续准入规则

1. 仅允许以下 ROM 进入仓库：
   - 自制 ROM（作者即仓库维护者）
   - 已明确授权可再分发的 ROM（含许可证文本）
   - 公有领域或明确自由许可 ROM（附来源与许可）
2. 每个保留 ROM 必须附带：
   - 来源链接
   - 许可证或授权文本
   - 授权核验日期
